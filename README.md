# Automação Cypress - E2E + API

## 📌 Objetivo
Automatizar fluxo de cadastro validando regras de negócio via UI e API.

## 🧱 Arquitetura
- Page Objects
- Custom Commands
- Utils (dados dinâmicos)
- Cenários orientados a dados

## 🔁 Estratégia de Testes
- UI: fluxo do usuário
- API: validação de backend
- Banco: validação de consistência

## ▶️ Execução
npm install
npx cypress open
npx cypress run# Automation-Exercise
Automação E2E com Cypress focada em cadastro de usuário, com validações de regras de negócio via UI e API. Estruturado com Page Objects, comandos customizados e dados orientados a cenário.
