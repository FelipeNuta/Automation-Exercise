# Automação Cypress - E2E + API

## 📌 Objetivo
Automatizar fluxo de cadastro validando regras de negócio via UI e API.

## 🧱 Arquitetura
- Page Objects
- Custom Commands
- Utils (dados dinâmicos)
- Cenários orientados a dados

## 🔁 Estratégia de Testes
- UI: fluxo do usuário
- API: validação de backend
- Banco: validação de consistência

## ▶️ Execução
npm install
npx cypress open
npx cypress run
